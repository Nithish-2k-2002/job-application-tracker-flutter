# Job Application Tracker 🚀

A modern Flutter application to manage and track job applications with AI chatbot support, dashboard analytics, smart suggestions, and clean UI.

---

# Features

✅ Add Job Applications  
✅ Smart Company Suggestions  
✅ Role Dropdown Selection  
✅ AI Career Chatbot  
✅ Dashboard Analytics  
✅ Modern UI Design  
✅ Responsive Layout  
✅ Search Bar  
✅ Flutter + Dart + SQLite  

---

# Tech Stack

| Technology | Usage |
|---|---|
| Flutter | UI Development |
| Dart | App Logic |
| SQLite | Local Database |
| Material UI | Modern Design |

# Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/job-application-tracker-flutter.git
```

---

# Open Project

```bash
cd job-application-tracker-flutter
```

---

# Install Dependencies

```bash
flutter pub get
```

---

# Run App

```bash
flutter run
```
